安裝完成，點選Next

![image](https://github.com/OreoKU/Crawler/blob/master/%E5%AE%89%E8%A3%9D7.png)

![image](https://github.com/OreoKU/Crawler/blob/master/%E5%AE%89%E8%A3%9D8.png)
